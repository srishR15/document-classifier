# document-classifier
## Classify Legal Documents

### 1. Run the following command:
```bash
pip install -r requirements.txt
```
### 2. Run the following command: 
```bash
python app.py
```

### 3. After the app runs, go to-
```bash
http://127.0.0.1:5000/
```
